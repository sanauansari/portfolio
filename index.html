UX Design · 2025–2026

Reduce Food Waste App

A two-phase UX project. Phase 1 (GDS 070) covered the full mobile experience — virtual fridge, pantry tracking, recipe suggestions, and QR scanning. Phase 2 (GDS 077) extended the design to Apple Watch, with expiry alerts, cooking timers, and chef coaching sessions.

UX Design Mobile Smartwatch Figma

Problem

Food is often forgotten in pantries and fridges and thrown away before it's used — people forget what they have, aren't sure if it's still safe to eat, or don't know how to use up leftovers and odd ingredients.

Solution

Tracks and helps use food before it expires — suggests recipes from available ingredients, sends expiration reminders, and shows a freshness guide for items in a virtual pantry and fridge. Phase 1 covered the mobile app; Phase 2 extended it to a smartwatch companion with quick-access reminders, recipe suggestions, and voice commands.

Target users

College students and young professionals, environmentally conscious consumers, parents involving kids in food prep, and roommates sharing groceries.

Key features

Voice command integration, user-defined push notifications, quick-glance fridge/pantry view, expiration reminders, shopping list sync, and voice-assisted cooking mode.

Spring 2026 · UX Design

Pixel Pioneers — Coding Adventure

Game-based learning app for 3rd graders. Block-based puzzles, mini game-building, and unlockable rewards. Meets California CS Standard 3-5.CS.2.

EducationMobileFigma
View case study

Background

Supports California Computer Science Standard 3-5.CS.2, teaching students how hardware and software work together through interactive coding activities and simple game-building lessons.

Objective

Introduces third-grade students to beginner coding concepts through fun, interactive, game-based learning experiences.

Target audience

3rd grade students, ages 7–8, beginner coding learners.

Key features

Block-based coding lessons, interactive puzzles and challenges, mini game-building activities, unlockable rewards and badges, kid-friendly visuals and animations, and a guide character with hints and support.

User task flow

Third grade students need a fun, hands-on, interactive way to explore basic coding concepts so they can develop problem-solving skills and create their own games.
