Pixel Pioneers — Coding Adventure

Education Mobile Figma

A game-based learning app introducing 3rd graders to coding through block-based puzzles, mini game-building activities, and unlockable rewards. Designed for GDS 072, Digital Imaging and User Interface Design, to meet California CS Standard 3-5.CS.2.

View full case study

Role

UX Designer — user task flow, wireframes, hi-fi prototype, and landing page.

Tools

Figma

Background

The app supports California Computer Science Standard 3-5.CS.2 by teaching students how hardware and software work together through interactive coding activities and simple game-building lessons.

Objective

To introduce third-grade students to beginner coding concepts through fun, interactive, and game-based learning experiences.

Target audience

  • 3rd grade students
  • Ages 7–8
  • Beginner coding learners

Key features

  • Block-based coding lessons
  • Interactive puzzles and challenges
  • Mini game-building activities
  • Unlockable rewards and badges
  • Kid-friendly visuals and animations
  • Guide character with hints and support

User task flow

Third grade students need a fun, hands-on, and interactive way to explore basic coding concepts, so they can develop critical problem-solving skills and create their own games.
